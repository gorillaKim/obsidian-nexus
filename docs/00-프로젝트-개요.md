# Obsidian Nexus

Tauri v2 기반의 에이전트 친화적 Obsidian 볼트 관리 시스템.

## 핵심 가치

1. **로컬 퍼스트** — 모든 데이터가 로컬에 저장, 외부 서버 불필요
2. **에이전트 친화적** — MCP 프로토콜로 AI 에이전트가 문서를 검색/탐색
3. **하이브리드 검색** — FTS5 키워드 + sqlite-vec 벡터 의미 검색
4. **멀티 볼트** — 여러 Obsidian 볼트를 통합 관리 및 검색

## 기술 스택

- **Backend:** Rust + Tauri v2
- **Frontend:** React + Tailwind CSS
- **DB:** SQLite (FTS5 + sqlite-vec)
- **Embedding:** Ollama (nomic-embed-text, 768차원)
- **CLI:** clap derive 기반
- **MCP:** stdin/stdout JSON-RPC

## 프로젝트 구조

```
obsidian-nexus/
├── apps/desktop/          # Tauri 데스크톱 앱
│   ├── src/               # React 프론트엔드
│   └── src-tauri/         # Rust 백엔드 (IPC)
├── crates/
│   ├── core/              # 핵심 엔진 (검색, 인덱싱, DB)
│   ├── cli/               # CLI 바이너리
│   └── mcp-server/        # MCP 서버 바이너리
└── docs/                  # 프로젝트 문서 (이 볼트)
```

## 볼트 설정 (on-config.json)

각 볼트 루트에 `on-config.json` 파일로 프로젝트 설정을 관리합니다.

```json
{
  "name": "Obsidian Nexus Docs"
}
```

- 볼트 등록 시 자동 생성 (없으면 폴더명으로 기본 생성)
- `name` 수정 후 `nexus_sync_config` 호출로 동기화
- `.obsidian` 폴더가 있는 디렉토리만 볼트로 인식
